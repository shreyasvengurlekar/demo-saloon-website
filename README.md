# 💇 B-Beyond Saloon Website

A modern and responsive website for **B-Beyond Saloon**.  
This project is designed to help customers easily explore salon services, prices, and contact information online.

---

## 📌 Project Overview

The **B-Beyond Saloon Website** provides an online platform for a salon where customers can:

- View available salon services
- Check price lists
- See salon gallery
- Book appointments (optional feature)
- Contact the salon easily

This website helps the salon attract more customers and build an online presence.

---

## 🚀 Features

- 🏠 Home page with attractive design
- 💇 Services section
- 💰 Price list display
- 🖼️ Salon gallery
- 📞 Contact section
- 📱 Fully responsive design (mobile friendly)

---

## ⚙️ Installation

1. Clone the repository

```

```

2. Open the folder

```
cd b-beyond-saloon
```

3. Open `index.html` in your browser.
